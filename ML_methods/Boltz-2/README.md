## Boltz-2

Boltz-2 is a machine learning scoring function designed to estimate protein-ligand binding affinity using statistical and structural descriptors derived from protein-ligand complexes. The model uses feature representations describing atom-level interactions between ligands and protein binding sites to predict binding strength.

In this benchmark study, Boltz-2 was used as an additional data-driven affinity predictor to evaluate how machine learning scoring functions compare with physics-based methods and deep learning models in virtual screening workflows.
