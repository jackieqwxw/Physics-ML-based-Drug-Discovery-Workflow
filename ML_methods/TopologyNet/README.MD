## \## TopologyNet

## 

## TopologyNet is a deep learning framework that uses topological data analysis to represent protein-ligand complexes. The model converts structural information into topological fingerprints derived from persistent homology and processes these features using convolutional neural networks.

## 

## In this study, TopologyNet was included to evaluate a machine learning approach that captures global structural patterns through topological descriptors rather than traditional geometric or physicochemical features.
