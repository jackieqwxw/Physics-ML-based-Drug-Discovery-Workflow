## RF-Score-VS

RF-Score-VS is a machine learning scoring function based on the Random Forest algorithm. It predicts protein-ligand binding affinity by using features derived from counts of atom-type contacts between ligands and protein residues within a specified distance cutoff.

RF-Score-VS has been widely used as a benchmark scoring function in virtual screening studies due to its simplicity and strong performance across diverse protein-ligand complexes. In this study, RF-Score-VS was included to evaluate the performance of classical machine learning scoring functions compared with deep learning and physics-based binding free energy methods.
