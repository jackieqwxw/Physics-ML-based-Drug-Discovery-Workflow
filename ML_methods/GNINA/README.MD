## GNINA

GNINA is a molecular docking program that integrates deep learning scoring functions with traditional physics-based docking algorithms. The method employs convolutional neural networks trained on protein-ligand complexes to improve scoring of docking poses and binding affinity predictions.

In this benchmark study, GNINA was evaluated as a hybrid approach combining docking and machine learning to predict binding free energy and discriminate active ligands from decoys.
